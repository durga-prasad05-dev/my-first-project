pseudocode is a step by step description of solution written in plain,human readble language




standard pseudocode structure

START
READ input
Initialize variables
Apply logic step  by step
Print output
END


EXAMPLE
ADD TWO NUMBERS

LOGIC 
1.TAKE 2 NUMBERS
2.ADD
3.DISPLAY RESULT

PSEUDOCODE

START
READ number1,number2
SET sum = number1 + number2
PRINT sum
END

EXAMPLE 2 DECISSION MAKING (IF-ELSE)
PROBLEM: CHECK EVEN OR ODD

LOGIC
1.IF NUMBER IS DIVISIBLE BY 2 THEN IT IS EVEN 
2.OTHERWISE IT IS ODD
   
PSEUDO CODE

START
READ number
IF number is %2 == 0
    Print "even"
ELSE
    Print "odd"
END
